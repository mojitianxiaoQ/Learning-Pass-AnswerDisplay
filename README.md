# Chaoxing-tool
本项目围绕超星学习通开发的辅助性脚本  
所有脚本均可以在篡改猴上使用，理论上所有js加载器均可以加载  
## 使用方法
#### 棱镜-学习通显示答案：  
这个脚本需要自行配置**deepseek-key**  
购买链接：[deepseek的API购买](https://platform.deepseek.com/usage)  
购买后将你的key填入js文件中的**配置区**  
打开到答题界面（必须打开到全览界面）点击左上角的按钮  
随后右侧将会出现一个小框框告诉你答案，拉到这个框的最底下就会出现一个一键答题的按钮  
一键答题有风险，效果非常暴力请谨慎使用  
按下左shift单击小框框中的题号可以查看本道题对应的题干（无选项）（该功能目前存在致命性错误无法使用）  
#### 学习通获取原题+答案：
仅当老师设置为你可以答题结束后查看答案并且你答题结束之后才可以使用该脚本  
打开到答案与题目共存的界面按下按钮即可导出csv文件  
#### 学习通获取题目原题
在正常你刚进入答题的时候，按下按钮即可进行题目导出，仅支持导出题目和选项，不支持导出答案  
**注意：以上三个脚本仅适用于纯本文题目，涉及到音频、图片均无法处理**
